# Sales Analysis Project

## Tools Used
- SQL
- Python
- Microsoft Excel
- Power BI
- DAX

## Project Overview
An end-to-end sales analytics project designed to transform raw sales data into actionable business insights. The project covers data cleaning, exploratory analysis, KPI reporting, dashboard development, and executive-level recommendations.

## Project Workflow
1. Cleaned and prepared sales data using SQL.
2. Performed exploratory data analysis using Python.
3. Built Pivot Tables and analytical reports in Excel.
4. Created visualizations and KPI tracking reports.
5. Developed an interactive Power BI dashboard using DAX measures.
6. Prepared an executive presentation with business insights and strategic recommendations.

## Key Insights
- Phones and Chairs were the top-selling sub-categories.
- Copiers generated the highest profit contribution.
- Tables and Bookcases were the main loss-driving categories.
- West and East regions delivered the strongest overall performance.
- Profitability opportunities were identified through pricing and cost optimization analysis.
