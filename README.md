# Sales Analysis Project

## Tools Used
- SQL
- Python
- Excel
- Power BI

## Project Overview
This project analyzes sales data using SQL, Python, Excel, and Power BI to identify trends, track KPIs, and generate business insights.

## Key Features
- Data Cleaning using SQL
- Exploratory Data Analysis using Python
- Pivot Tables and Charts in Excel
- Interactive Dashboard in Power BI
- DAX Measures and KPI Tracking
