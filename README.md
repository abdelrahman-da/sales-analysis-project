# Sales Analysis Project

## Tools Used

* SQL
* Python
* Microsoft Excel
* Power BI
* DAX

## Business Problem

The company needed to understand sales performance, profitability drivers, regional trends, and loss-generating product categories to support data-driven business decisions.

## Project Overview

An end-to-end sales analytics project designed to transform raw sales data into actionable business insights. The project covers data cleaning, exploratory analysis, KPI reporting, dashboard development, and executive-level recommendations.

## Project Workflow

1. Cleaned and prepared sales data using SQL.
2. Performed exploratory data analysis using Python.
3. Built Pivot Tables and analytical reports in Excel.
4. Created charts and KPI tracking reports.
5. Developed an interactive Power BI dashboard using DAX measures.
6. Prepared an executive presentation with business insights and recommendations.

## Key KPIs

* Total Sales: $2.3M
* Total Profit: $286.76K
* Total Orders: 9,994
* Total Quantity Sold: 37,873
* Average Profit per Order: $28.7

## Key Insights

* Phones and Chairs were among the top-selling product categories.
* Copiers generated the highest profit contribution.
* Tables and Bookcases showed lower profitability compared to other categories.
* West and East regions delivered the strongest overall sales performance.
* Business opportunities were identified through sales and profitability analysis.
