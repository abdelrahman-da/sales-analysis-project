# Sales Analysis Project

## Tools Used

* SQL
* Python
* Microsoft Excel
* Power BI
* DAX

## Business Problem

The objective of this project was to analyze sales performance, identify profitability drivers, detect loss-generating categories, and provide data-driven recommendations to improve business performance.

## Project Overview

This end-to-end Sales Analysis project combines SQL, Python, Excel, and Power BI to transform raw sales data into actionable business insights. The project includes data cleaning, exploratory data analysis, KPI reporting, dashboard development, and business recommendations.

## Project Workflow

1. Cleaned and prepared sales data using SQL.
2. Performed exploratory data analysis using Python.
3. Created Pivot Tables and analytical reports in Excel.
4. Built charts and KPI reports for business presentation.
5. Developed an interactive Power BI dashboard using DAX measures.
6. Presented business insights and recommendations based on the analysis.

## Key KPIs

* Total Sales: $2.3M
* Total Profit: $286.76K
* Total Orders: 9,994
* Total Quantity Sold: 37.87K
* Average Profit: $28.7

## Key Insights

* Phones and Chairs were the highest-selling sub-categories.
* Copiers generated the highest profit among all sub-categories.
* Tables and Bookcases were the main loss-generating categories.
* The West and East regions achieved the strongest overall performance.
* Technology was the top-performing product category by sales.
